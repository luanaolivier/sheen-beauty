SHEEN BEAUTY

Projeto autoral de e-commerce fictício desenvolvido com foco em identidade visual, experiência do usuário e estruturação de interface front-end.



//CONTEXTO

O mercado de cosméticos é altamente visual e competitivo, exigindo interfaces que transmitam sofisticação, organização e confiança. O projeto Sheen Beauty foi criado como um estudo prático de construção de marca digital e desenvolvimento de interface para e-commerce, unindo UX/UI Design e implementação em HTML e CSS.



//OBJETIVO DO PROJETO

- Criar uma identidade visual autoral para uma marca fictícia de cosméticos.

- Desenvolver uma interface moderna e responsiva.

- Aplicar princípios de hierarquia visual e organização de produtos.

- Simular uma experiência de navegação fluida e intuitiva.



//PROCESSO
1️- Identidade Visual: Desenvolvimento de uma identidade visual própria, definição de estilo, paleta de cores e posicionamento da marca.

2️- Prototipação no Figma: Estruturação do layout, Organização da vitrine de produtos, Definição de hierarquia visual, Planejamento da navegação. 

3️- Implementação: Estruturação semântica com HTML5, Estilização com CSS3, Organização do código em seções claras, Ajustes de responsividade.



//TECNOLOGIAS ULTILIZADAS

- HTML5

- CSS3

- Figma 



//FUNCIONALIDADES

- Layout responsivo

- Estrutura organizada por seções (hero, categorias, vitrine, rodapé)

- Organização visual focada em clareza e escaneabilidade

- Simulação de vitrine de produtos



//APRENDIZADOS TÉCNICOS E ESTRATÉGICOS 

- Transformar identidade visual em interface funcional.

- Aplicar conceitos de UX na organização de e-commerce.

- Estruturar código front-end de forma limpa e escalável.

- Entender a relação entre estética e usabilidade.
